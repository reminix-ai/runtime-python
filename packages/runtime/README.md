# reminix-runtime

Core Reminix Runtime for deploying AI agents via REST APIs.

## Installation

```bash
pip install reminix-runtime
```

## Usage

```python
from reminix_runtime import serve, BaseAdapter

# Serve wrapped agents
serve([wrapped_agent], port=8080)
```
